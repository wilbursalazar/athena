# User Stories

1. I'm a visitor who wants to learn how to donate money so I can help families get books and toys. I should be able to submit my contact information through a form.
2. I'm a parent who wants to learn how the nonprofit works and request books and toys. I should be able to understand Athena's mission to see if it's a good fit.
3. I'm a library, shelter, or church representative who wants to apply to become a bulk donor of books and toys. I should be able to submit my address, contact information, and donation type via a form.
